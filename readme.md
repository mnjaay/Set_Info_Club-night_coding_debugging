Bug Hunting – Compétition du Club Informatique

Bienvenue à la Partie  : Correction de Bugs - Night Code du Club Informatique !
Dans cette activité, vous allez travailler en équipe pour analyser, comprendre et corriger des algorithmes en langage C.

L’objectif est de tester vos compétences en débogage, en compréhension de code, et en collaboration.

Contenu du dépôt

Ce dépôt contient 3 niveaux de difficulté :

1. Niveau Facile

Objectif : repérer des erreurs évidentes (boucles incorrectes, mauvais format d’affichage, absence de return, etc.)

Contient :

facile_1.c

facile_2.c

2. Niveau Intermédiaire

Objectif : corriger des erreurs plus subtiles 

Contient :

intermediaire.c

3. Niveau Difficile

Objectif : corriger des bugs invisibles à l’exécution mais qui faussent le résultat 

Contient :

difficile.c

Comment récupérer les fichiers ?
1. Cloner le dépôt
git clone https://github.com/<nom-du-club>/bug-hunting.git

2. Entrer dans le dossier
cd nuit-coding-debugging

Créer une nouvelle branche pour votre équipe
1. Exemple creation branche 
git checkout -b equipe1
2. Vérifier sur quelle branche vous êtes
Modifier les fichiers et corriger les bugs

Les équipes corrigent les codes dans leur dossier correspondant :
