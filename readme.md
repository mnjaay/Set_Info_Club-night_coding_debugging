Bug Hunting – Compétition du Club Informatique

Bienvenue à la Partie : Correction de Bugs - Night Code du Club Informatique ! Dans cette activité, vous allez travailler en équipe pour analyser, comprendre et corriger des algorithmes en langage C.

L’objectif est de tester vos compétences en débogage, en compréhension de code, et en collaboration.

Contenu du dépôt

Ce dépôt contient 3 niveaux de difficulté :

Niveau Facile
Objectif : repérer des erreurs évidentes

Contient :

``` facile_1.c ```

``` facile_2.c ```

``` facile_3.c ```

facile_3.c

Niveau Intermédiaire
Objectif : corriger des erreurs plus subtiles

Contient :

``` intermediaire.c ```

Niveau Difficile
Objectif : corriger des bugs invisibles à l’exécution mais qui faussent le résultat

Contient :

``` difficile.c ```

Comment récupérer les fichiers ?

Cloner le dépôt 

###
```bash
git clone https://github.com/mnjaay/Set_Info_Club-night_coding_debugging.git
```
###
